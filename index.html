параметры игрока

Кто использует аналитику?

Главный тренер

Резюме игры, анализ голевых моментов, сочетания пятерок, индивидуальный анализ игроков, поиск слабых мест соперника, входы/выходы из зоны

Генеральный менеджер

Выявление недооцененных и переоцененных игроков на рынке, экономия бюджета клуба на неэффективных контрактах и недооценённых игроках

Старший тренер

Позиционирование/тактика, пасы, владение шайбой, блокированные броски, игра в защите (подавление голевых моментов соперника)

Помощник тренера

Анализ противника, анализ игры в неравных составах, анализ игрока по: темпу, скорости, ускорениям, максимальной скорость, интенсивности катания

Тренер по физ. подготовке

Игровой ритм, процесс восстановления, сезонные оценки игроков, кондиция игрока на площадке, оптимальная длительность смены для каждого игрока

Тренер вратарей

Анализ всех бросков по воротам во время матча, разделение бросков по зонам, идентификация слабых мест вратаря, контроль отскоков

Скаутинг

Анализ существующих пробелов в составе; выявление потенциальных игроков на рынке, отвечающим вашим потребностям, сравнительный анализ игроков

Скаутинг юниоров

Рассмотрение игрока в качестве партнеров/соперника по команде, оценка: бросков, пас, хоккейный IQ, усердие, обращение с клюшкой, оборона, позиционирование
